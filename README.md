# Price Of A Launch - SpaceX Falcon 9 Analysis
This project analyzes SpaceX Falcon 9 launch data to predict first-stage landing success and estimate launch costs using data science and machine learning techniques.

## Objectives
- Collect SpaceX launch data
- Perform data wrangling and cleaning
- Analyze launch trends using SQL and visualization
- Build predictive machine learning models
- Create an interactive dashboard
- Visualize launch locations using maps

## Repository Contents
- `spacex-data-collection-api.ipynb` → Data collection from API
- `Spacex-Data wrangling-v2.ipynb` → Data cleaning
- `SpaceX EDA with SQL.ipynb` → SQL analysis
- `Spacex Complete the EDA with Visualization.ipynb` → Visual exploration
- `SpaceX-Machine-Learning-Prediction...ipynb` → Prediction models
- `spacex-dash-app.py` → Interactive dashboard
- `Launch Sites Locations Analysis with Folium.ipynb` → Geospatial mapping

## Technologies Used
- Python
- Pandas
- NumPy
- SQL
- Plotly Dash
- Folium
- Scikit-learn
- Jupyter Notebook
